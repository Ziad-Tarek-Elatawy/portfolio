💼 2. نسخة مستقل/خمسات (فصحى احترافية — Case Study للبورتفوليو)
العنوان: بناء نظام أتمتة ذكي (Automation System) لمراقبة البيانات وإرسال تنبيهات لحظية عبر Telegram

نظرة عامة على المشروع: في عالم الأعمال الحديث، سرعة الاستجابة للفرص هي العامل الحاسم. تم بناء هذا النظام لأتمتة عملية "مراقبة الفرص والمشاريع" يدوياً، وتحويلها إلى نظام ذكي يعمل على مدار الساعة ويقوم بإرسال إشعارات فورية عبر Telegram فور ظهور بيانات تطابق معايير محددة.

التحديات التي تم حلها:

المراقبة دون توقف (24/7): بناء مسار عمل (Workflow) يقوم بجلب البيانات كل 5 دقائق دون أي تدخل بشري.
تخطي الحماية الأمنية (Stealth Mode): برمجة آلية لتغيير الـ User-Agents وإضافة تأخيرات زمنية عشوائية لمحاكاة السلوك البشري وتخطي أنظمة مثل Cloudflare.
معالجة النصوص المعقدة (Data Parsing): استخدام تقنيات Regex لفك تشفير الروابط العربية (URL Encoding) واستخراج العناوين والروابط بشكل دقيق من الكود المصدري للموقع.
منع التكرار (Deduplication): بناء ذاكرة مؤقتة (7-Day TTL) تضمن عدم إرسال نفس الإشعار مرتين مطلقاً.
القيمة المضافة للعمل (ROI):

توفير 100% من وقت البحث اليدوي: تحويل عملية تستغرق ساعات يومياً إلى عملية مؤتمتة بالكامل.
استجابة فورية: الحصول على المعلومة في نفس ثانية نشرها.
التقنيات والأدوات: n8n (Workflow Automation) · Web Scraping · Telegram Bot API · Advanced Regex · JSON Parsing

🌍 3. نسخة Upwork/Freelancer (Professional English — Global Market)
Project Title: Automated Stealth Monitoring System & Telegram Lead Sniper

Project Overview: In highly competitive markets, response time defines success. Waiting to manually refresh a webpage means losing the lead. I engineered a fully autonomous web-monitoring pipeline using n8n that acts as a 24/7 lead sniper. The system silently scrapes target platforms, parses the raw HTML, applies advanced filtering rules, and dispatches real-time, easily readable alerts directly to a Telegram Bot.

Key Technical Achievements:

Stealth Browsing Engine: Designed a randomized User-Agent rotation and human-like delay system to successfully bypass strict anti-bot protections (e.g., Cloudflare) without relying on expensive proxy APIs.
Advanced Data Extraction: Utilized complex Regular Expressions (Regex) to extract hidden relative URLs and dynamically decode complex URL-encoded strings into readable text.
Intelligent Deduplication: Engineered a stateful memory system (7-Day TTL Cache) within the workflow to ensure zero duplicate alerts are sent.
Fail-Safe Integrity: Built-in error handling that detects HTTP 403/503 blocks, pauses the execution, and alerts the user of the system's status before auto-retrying.
Business Value Delivered:

Eliminated hours of manual data-entry and active monitoring.
Granted a massive "first-mover" advantage by ensuring the user is notified within minutes of a target event occurring.
Tech Stack: n8n (Automation) · JavaScript · Telegram API · HTML DOM Parsing · System Architecture
