Title: End-to-End Machine Learning Web App for Real Estate Price Prediction

Description: "I designed and developed a complete, end-to-end Machine Learning pipeline to predict Boston housing prices based on various socio-economic features.

Starting with the raw dataset, I performed comprehensive Exploratory Data Analysis (EDA) and feature engineering using Pandas and NumPy. For the machine learning phase, I trained a highly optimized Decision Tree Regressor. To ensure robust performance and prevent overfitting, I utilized Grid Search Cross-Validation (GridSearchCV) combined with ShuffleSplit, fine-tuning hyperparameters to lock in the optimal R-squared accuracy score.

Instead of stopping at an offline Jupyter Notebook, I took this project to production. I engineered a lightning-fast, asynchronous REST API using FastAPI to securely serve the serialized ML model. Finally, I built a modern, fully responsive web frontend (using HTML/CSS and Jinja2) featuring a sleek glassmorphism UI, allowing end-users to input features and receive real-time, accurate property valuations.

💡 Core Technologies Used:

Machine Learning: Python, Scikit-learn, Pandas, NumPy
Backend / API: FastAPI, Uvicorn, Jinja2
Frontend: HTML5, CSS3 (Modern UI/UX Design)
DevOps / Version Control: Git & GitHub
